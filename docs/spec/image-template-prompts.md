# 封面与内文模板生图提示词

## 通用要求

每张图必须严格使用指定像素尺寸。模板只生成背景，后续由程序叠加中文文字：封面使用主标题和
可选副标题，内容图使用主标题和可选正文。
所有提示词末尾统一追加：

> reusable social-media template background, clean low-detail text-safe area, balanced composition, production quality, no text, no letters, no numbers, no logo, no watermark, no UI mockup, no phone frame

生成后检查：

- 指定文字区域没有人物、物品、强纹理或高对比装饰。
- 图片中不存在任何文字、字母、数字、Logo、水印和伪文字。
- 不要生成手机壳、平台界面或封面展示样机。
- 装饰不得越过指定安全区，画面四周不要留额外白边。
