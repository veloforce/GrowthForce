# Template fonts

The image template renderer bundles these JiangChengZhiYinTi font files for deterministic cross-platform rendering:

- `JiangChengZhiYinTi-600W.ttf`: cover titles, content-card titles, and cover subtitles.
- `JiangChengZhiYinTi-400W.ttf`: content-card body text.

The renderer intentionally does not fall back to operating-system fonts. Missing bundled fonts are treated as an incomplete installation.

JiangChengZhiYinTi is distributed under the SIL Open Font License 1.1. Keep `OFL-1.1.txt` with the font files in source and release packages.
