# 内容研究

本 Skill 负责形成可供策略和创作消费的研究结果，不负责选题决策、写作或长期沉淀。
本 Skill 返回研究结果供策略和创作阶段消费；不创建 Run、不写长期数据。需要保存到 Run
`research` 文档时，由发起的生成/发布阶段写入。

## 输入与输出

- 输入：平台、账号定位、研究目标、参考账号、当前工作目录素材和用户限制。
- 输出：热点、竞品、事实来源、证据缺口和建议研究方向。
- 外部来源必须记录 URL、访问时间和摘要；不得把模型记忆表述为已验证事实。

## 路由

- 泛热点、关键词热点和公众号竞品候选使用 `content_research` 工具。
- 具体网页和补充事实使用 browser 工具。
- 小红书站内搜索、详情和主页相关，请使用 references/xhs.md
